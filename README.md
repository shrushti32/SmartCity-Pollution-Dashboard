# 🌆 Smart City Pollution Monitoring Dashboard (SDG-11)

## 📊 Overview

This project presents an interactive **Power BI dashboard** for monitoring air pollution levels across different cities.
It helps analyze **AQI (Air Quality Index)** and key pollutants to support sustainable urban planning.

---

1.  Features

* 🌫️ AQI (Air Quality Index) Monitoring
* 📊 Pollutant Analysis (PM2.5, PM10, NO2, CO, Ozone)
* 🌍 City & Country Filters
* 📈 Trend Analysis Over Time
* ⚠️ Pollution Alert System

---

2.  Tools Used

* Microsoft Power BI
* Data Visualization
* Air Pollution Dataset

---

3.  Dataset

The dataset contains information about:

* City & Country
* AQI Values
* PM2.5, PM10, NO2, CO, Ozone Levels
* Date-wise Pollution Data
