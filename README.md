 Smart City Pollution Monitoring Dashboard (SDG-11)

Overview

   This project presents an interactive Power BI dashboard for monitoring air pollution levels across cities.

2.  Features

  * AQI (Air Quality Index) Monitoring
  * Pollutant Analysis (PM2.5, PM10, NO2, CO, Ozone)
  * City & Country Filters
  * Trend Analysis over Time
  * Pollution Alert System

3.  Tools Used

   * Power BI
   * Python / Excel Dataset

4.  Dashboard Preview

   [Dashboard](dashboard.png)

5.  Objective

To support smart city planning and improve environmental monitoring aligned with SDG-11.

6.  How to Use

  * Download the `.pbix` file
  * Open using Power BI Desktop
  * Explore dashboard using filters
